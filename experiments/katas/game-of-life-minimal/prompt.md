# Game of Life Kata

## Task

Implement Conway's Game of Life on an infinite grid. Given a set of living cells, compute the next generation.

## Expected Output Files

- `src/game-of-life.ts` - Implementation
- `src/game-of-life.spec.ts` - Tests

## Constraints

- Use Vitest for testing
- Use TypeScript
- Use sparse representation (only store living cells)
- Handle negative coordinates
