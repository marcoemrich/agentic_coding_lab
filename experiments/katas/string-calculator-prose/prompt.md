# String Calculator Kata

## Feature

The String Calculator is a small utility with a single `add` function that takes a string of numbers and returns their sum as an integer. The numbers in the input string are separated by commas. If the input string is empty, the result is zero. A string containing a single number returns that number. If the string contains two or more numbers separated by commas, the function returns the sum of all of them.

## Task

Implement the `add` function according to the description above.

## Expected Output Files

- `src/string-calculator.ts` - Implementation
- `src/string-calculator.spec.ts` - Tests

## Constraints

- Use Vitest for testing
- Use TypeScript
