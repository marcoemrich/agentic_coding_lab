# Word Score Kata

## Task

Implement a function that calculates the score of a word by summing the alphabet positions of its letters (A=1, B=2, ..., Z=26), case-insensitively.

## Expected Output Files

- `src/word-score.ts` - Implementation
- `src/word-score.spec.ts` - Tests

## Constraints

- Use Vitest for testing
- Use TypeScript
- Input contains only letters (A-Z, a-z) or empty string
- No spaces, numbers, or special characters in input
