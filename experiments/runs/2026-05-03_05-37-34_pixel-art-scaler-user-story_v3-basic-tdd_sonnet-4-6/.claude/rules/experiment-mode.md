# TDD Experiment Mode

## Instructions

Complete the TDD exercise **autonomously** using Test-Driven Development.

- Do NOT wait for human approval between phases
- Run the full TDD cycle until all tests pass

## Running Tests

Use `pnpm test` to run tests.

## Done Marker

When you are finished, write a file `experiment-done.txt` with the single word `DONE` as its only content. Do not write any other summary or report file.
