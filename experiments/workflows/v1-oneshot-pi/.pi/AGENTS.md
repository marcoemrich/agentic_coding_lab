# One-Shot Experiment Mode (pi)

Complete the coding exercise in a single pass. Read `prompt.md`, then implement the solution directly.

- Do NOT wait for human approval; complete the task autonomously.
- Do NOT use TDD; write the implementation directly.

## Approach

1. Read the requirements from `prompt.md`.
2. Implement the solution.
3. If the prompt asks for a CLI entry point, create `src/cli.ts` (reads JSON from stdin, writes JSON to stdout).
4. Run any verification you find appropriate.

## Done Marker

When you are finished, write a file `experiment-done.txt` whose entire content is the single word `DONE`. Do not write any other summary or report file.
